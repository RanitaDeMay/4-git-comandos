Instalación y configuración

config

Obtiene y configura el repositorio o las opciones globales.

git config

Parámetros

Parámetro Función Ejemplo
--global Configuración global de nombre de usuario y dirección de correo electrónico. git config --global user.email "email@gmail.com"
--add Agregar una configuración específica para un repositorio git config --add branch.master.name "main"
--e Editar la configuración existente git config --global --edit
